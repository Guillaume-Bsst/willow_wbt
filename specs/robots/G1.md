# Unitree G1 — Robot Specification

## Overview

| Property | Value |
|----------|-------|
| Manufacturer | Unitree Robotics |
| Type | Bipedal humanoid |
| DOF modes | 29-DOF (default), 27-DOF |
| Height | ~1.27 m |
| Simulator | MuJoCo (inference/eval), IsaacGym / IsaacSim / MJWarp (training) |
| Base link | `pelvis` |

---

## DOF Modes

### 29-DOF (default)

| URDF | `g1_29dof_retargeting.urdf` |
|------|---------------------------|
| Waist joints | `waist_yaw_joint`, `waist_roll_joint`, `waist_pitch_joint` |

### 27-DOF

| URDF | `g1_27dof_retargeting.urdf` |
|------|---------------------------|
| Waist joints | `waist_yaw_joint` only (roll and pitch removed) |

Use `--robot-config.robot-dof 27` or `29` to select the mode.

## qpos Convention (MuJoCo)

| Indices | Content |
|---------|---------|
| `0:3` | Base position `[x, y, z]` in metres |
| `3:7` | Base quaternion `[qw, qx, qy, qz]` (**wxyz**) |
| `7:36` | Joint positions (29 DOFs), radians |

Total size: `7 + 29 = 36` for 29-DOF mode, `7 + 27 = 34` for 27-DOF.

## Scale Factor

For retargeting, human motion is scaled to match robot height:

```
scale = ROBOT_HEIGHT / human_height
ROBOT_HEIGHT = 1.27  # metres
```
